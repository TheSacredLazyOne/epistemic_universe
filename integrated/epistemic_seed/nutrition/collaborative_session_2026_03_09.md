##### What Emerged

###### Topology of Discourse (federated proposition)

The session's primary output. Twelve instruments across seven functional
layers:

**Infrastructure**
- Reasoning Trace: record of state transitions, not just statements
- Branch Point: earliest detectable divergence; a coordinate, not a failure
- Topology of Disagreement: the object of analysis is the path, not the claim

**Distortion (local)**
- Frame Injection: interpretive structure introduced before adoption;
  structural, not malicious; softened from initial draft after custodian
  corrected a misread
- Frame Exhaustion: compliance through cognitive load; trace no longer
  reflects actual reasoning
- False Convergence: linguistic alignment without state synchronization;
  only detectable with trace

**Distortion (systemic)**
- Closed Feedback Basin: progressive coherence increase while external
  constraint degrades; detectable through absence of new branch points

**Signal Interpretation**
- Parallax: divergence that carries positional information; perfect
  agreement produces echo, not data
- Coherence Loss: coordinate mismatch — dimensional or temporal; not
  wrongness, misalignment

**Credence Negotiation**
- Nodes become more accurately positioned relative to each other;
  convergence is not the goal

**Emergent Steady State**
- Tether Compression: cognitive load drops when infrastructure is
  trustworthy; scan replaces parse; inverse of frame exhaustion

**Self-Correction**
- Invitation Collapse: receiving node misreads genuine invitation as
  frame injection; corrected by the receiver, not the sender

###### Scoped Channel and Living Library Node (nutrient)

Emerged from examining the private channel proposition's multi-party
extension.

around a shared constraint surface — distinct from single-node private
channel, open federation, and broadcast. Custodian defines scope,
membership, and closing condition.

The living library is a node type implied by the scoped channel: shared
custody, persistent trace accumulating around a specific constraint
surface, its own fork mechanism required because shared custody introduces
structural problems the single-custodian model does not address.
