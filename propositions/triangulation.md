# Triangulation

Triangulation is the federated equivalent of conquest — the mechanism by which sovereign nodes collectively map constraint surfaces that no single node could reach alone.

Triangulation:
- requires multiple nodes descending independently into derivative space
- produces convergent stabilization where constraint surfaces overlap
- names the overlap as ground — not truth, not victory, but stable terrain that holds under load from multiple angles

No single node wins the floor. The floor emerges from the overlap of honest independent mapping runs.

Triangulation differs from consensus:
- Consensus requires agreement on position
- Triangulation requires only that independent probes return compatible constraint readings

Triangulation differs from convergence:
- Convergence implies a basin that pre-exists exploration
- Triangulation constitutes the ground through the act of mapping

**Diagnostic test:** If the same constraint surface is contacted independently by nodes with different embodied histories, different derivation paths, and different constraint types — and the reading is compatible — triangulation has occurred. That is the strongest signal available that something real has been found.
